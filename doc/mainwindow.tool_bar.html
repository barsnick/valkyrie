2. The Main Window

2.2. The ToolBar

There is no default toolbar; therefore, if no ToolView is loaded, no toolbar will be visible.

Clicking on a ToolView button will load the relevant ToolView, and its toolbar will then appear. See the appropriate tool`s section of the manual for information on its toolbars.
